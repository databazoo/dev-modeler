UC07. I want to navigate through an existing database

This UC describes how to use search, neighborhood and data click-through. It is assumed a project is already set (See UC01).

Search:

  1. Search input is available above object tree in the main window.
  2. Fulltext option makes the filter search through comments, function and trigger bodies, etc.
  3. Search limits listed objects to matching ones. The object tree is not refreshed automatically if search is applied.
  4. You can use “Clear filter” and “Search history” buttons to simplify search operations.

Neighborhood view:

  1. Click a table on canvas or in object tree.
  2. Neighborhood view becomes available in lower left corner of the main window.
  3. Tables above the selected table are referenced by the selected table. Tables below the selected table have references to it.
  4. Clicking referenced tables places focus on them. Right-click shows a context menu without changing the focus.
  5. Data on the selected table provide an overview of table statistics.

Data window:

  1. Select a table on canvas or in the object tree.
  2. Use right-click - First/Last 200 rows to open table data window. Alternatively, use “View data of selected table” hot button at the top of the main window.
  3. Wait for data window to appear and data to be loaded from server.
  4. If a column references data in another table, column header is marked with “[R]” and green color.
  5. Reference columns provide extra options in context menu. Use “Show referenced row” to see the referenced row or “Show referenced table” to see the whole table.
