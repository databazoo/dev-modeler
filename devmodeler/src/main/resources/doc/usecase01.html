UC01. I want to connect to an existing database

This UC describes how one can connect to an existing database, creating a new project on the way.

  1. Go to menu File - Projects.
  2. Select “Create a new project” or press “New” button.
  3. In simple mode:
    1. Fill in “Project name”.
    2. Select database type.
    3. Fill in “Host”, “User” (optional) and “Password” (optional).
    4. If access information is correct a list of available databases will be shown. Choose one or more and press OK.
    5. Press “Create” button and wait until database information is loaded from server.
  4. In advanced mode:
    1. Select database type and press “Next”.
    2. Choose appropriate server count.
    3. Fill in connection information for every environment listed in “Server connections” table. Use “Delete” key to remove unneeded rows. Press “Next”.
    4. If access information is correct for the first environment a list of available databases will be shown. Choose one or more and press OK.
    5. Fill in “Project name”.
    6. Choose “Store revisions to” path. Changes you make will be stored there in XML format. If you already use VCS for your application source files you can choose your database changes to share same location.
    7. You may also use “Configure separate setting for each database and connection” checkbox at the bottom of the window to finetune your connection settings for different environments. Upon checking a table will be shown where you can set different aliases for same database throughout the environments, different access info, etc.
    8. Press “Create project” button and wait until database information is loaded from server.
  5. You may want to use menu Model - Rearrange items to arrange downloaded objects automatically on the canvas. Try different options for better results and finetune placements manually.

NOTE: “Host” input fields may contain full resource information including port (:5432), Oracle SID (:SID or /serviceName), MSSQL instance (\instance), etc. See JDBC connection string documentation for your target DB type.
