UC12. I want to see running processes

This UC describes how one can list active processes from given server. It is assumed a project is already set (See UC01).

  1. Select target environment in upper right corner of the main window.
  2. Press “Process list” button.
  3. If you want to terminate a long-running process (marked with red):
    1. Press “Pause”.
    2. Select a process.
    3. Press “Cancel” (stop the process) or “Terminate” (kill the process).
    4. Press “Pause” to enable autorefresh again.

NOTE: Operations on process list depend on database type.

NOTE: To terminate a process you must be given rights to do so.
