UC11. I want to compare 2 databases

This UC describes how to compare schema of 2 databases. It is assumed a full development stack is set (See UC03).

  1. Switch view using “Changes” button at the top of the main window.
  2. Switch to “Compare databases” tab.
  3. Choose appropriate environments to compare.
  4. Press “Compare” and wait for database information to be loaded from servers.
  5. Double-click individual listed differences to show side-by-side SQL comparison.
  6. Accept one of the offered versions with “Accept (apply to…)” button to resolve the difference.

NOTE: Compare tool should only be used to detect older differences. It should never be used to publish changes between environments. For this purpose use revisions (See UC10).
