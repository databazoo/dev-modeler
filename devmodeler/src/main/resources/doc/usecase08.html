UC08. I want to optimize SQL queries

This UC describes how one can optimize application queries (like ones appearing in slowlog). It is assumed a project is already set (See UC01).

  1. Press “Run SQL queries” hot button at the top of the main window.
  2. Insert your query into SQL input.
  3. Press “Explain” or hit “F6” to show operation tree.
  4. Operation tree shows individual operations as they would be executed on the server.
  5. Lines connecting operations show the operation load. Thicker lines mean heavier operations.
  6. Click the operation to show operation details.
  7. Choose “Data” tab on the right to see the explanation in text mode.
  8. According to information you find in the explanation, alter your query or add indexes.
  9. Rerun the explain with “Explain” button or hit “F6”.

NOTE: availability of operation properties depend on your database type and server version.
