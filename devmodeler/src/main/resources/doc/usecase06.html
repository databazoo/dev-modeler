UC06. I want to see, export and edit data in a table

This UC describes how one can select, update and delete data from a table. It is assumed a project is already set (See UC01).

  1. Select a table on canvas or in the object tree.
  2. Use right-click - First/Last 200 rows to open table data window. Alternatively, use “View data of selected table” hot button at the top of the main window.
  3. Wait for data window to appear and data to be loaded from server.
  4. Use context menu options to narrow down your search. Right-click a value in the table and choose one of filtering and ordering options.
  5. Double-click a value to edit it. Hit “Ctrl+S”, “Tab” or “Enter” keys to save or “Escape” key to cancel the edit.
  6. Use last (empty) line of the table to insert a new row. Confirm the insert with “Ctrl+S”.
  7. Use context menu option “Delete row” to delete a row or multiple rows. You will be asked to confirm deletion.
  8. Use “Export” button at the upper left corner of the table to export current data. You can choose from XLS or CSV formats.

NOTE: data operations are not available on tables without primary keys, views and procedure results.

Tabular data

When altering the SQL query, you might want to insert tabular data from Microsoft Excel, OpenOffice/LibreOffice Calc or result data tables. This is supported by formatting inserted data in 2 modes:

  1. Single-column data is transferred to comma-separated list for use in constructs like WHERE ... IN (1,2,3...)
  2. Multi-column data is formatted to fit the needs of constructs like INSERT ... VALUES (1,2,3...), (4,5,6...)

NOTE: if the automatic change does not suit your case, simply hit “Ctrl+Z” to get back to raw insert.
