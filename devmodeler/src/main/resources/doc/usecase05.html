UC05. I want to create a table

This UC describes how one can create a table. It is assumed a project is already set (See UC01).

  1. Right-click a schema and choose “Add new table”.
  2. Fill in “Name”.
  3. Press “Save” or hit “Ctrl+S”.
  4. If the database type does not require at least one column:
    1. Select “New attribute” in the tree.
    2. Fill in “Name” with “id”.
    3. Fill in “Datatype” with “serial” (PostgreSQL) or “integer”.
    4. Fill in “Default” with “autoincrement” (MySQL).
    5. Press “Save” or hit “Ctrl+S”.
  5. If the database type does not require a primary key:
    1. Select “New primary key” in the tree.
    2. Press “Save” or hit “Ctrl+S”.
  6. Feel free to add more columns or other objects (indexes, unique keys, etc.).
  7. Close the wizard window.
